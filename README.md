# clean_architecture_reactjs

### Dependencies
`
npm install --prefix ./frontend/web ./frontend/core ./backend
`

### Usage
#### Start project
```
./dev.sh
```
or
```
npm run develop --prefix ./backend
npm run dev --prefix ./frontend/web
```
#### Open project
```
https://192.168.0.1/3000
https://192.168.0.1/1337
```
